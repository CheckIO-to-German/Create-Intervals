You have a set of ints. You should create a list of intervals out of it.

Input: A set of ints.

Output: A list of tuples. Each tuple has two elements, both are int. The elements show beginning and ending of interval.

Example: 

create_intervals({1, 2, 3, 4, 5, 7, 8, 12}) == [(1, 5), (7, 8), (12, 12)]
create_intervals({1, 2, 3, 4, 5, 6, 7, 8}) == [(1, 8)]
